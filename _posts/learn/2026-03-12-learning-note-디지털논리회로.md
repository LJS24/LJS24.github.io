### 1) 진수법(진수 표현법)
- 진법을 나타내는 기본 수를 기수(radix)라고 칭하며, 이 기수가 진법의 이름이 된다.
- 10진수 표현법:
    - 기수가 10이며, 0~9의 10개 수를 이용하여 표현하는 진법.
    - 어떤 수의 각 자리의 수를 &nbsp; $n*10^m$ (10의 거듭제곱이 곱해진 형태)으로 표현. \\
    예시: $123 = 1 * 10^2 + 2 * 10^1 + 3 * 10^0$
- 2진수 표현법:
    - 기수가 2이며, 0과 1의 2개 수로 표현하는 진법.
    - 어떤 수의 각 자리의 수를 &nbsp; $n*2^m$(2 의 거듭제곱이 곱해진 형태)으로 표현.
- 8진수 표현법:
    - 기수가 8이며, 0~7의 8개 수로 표현하는 진법.
    - 어떤 수의 각 자리의 수를 &nbsp; $n*8^m$(8 의 거듭제곱이 곱해진 형태)으로 표현.
- 16진수 표현법:
    - 0~9, A~F까지 16개의 기호로 표현.
    -  어떤 수의 각 자리의 수를 &nbsp; $n*16^m$(16 의 거듭제곱이 곱해진 형태)으로 표현.
- 진수법 비교:
    - 2진수의 3자리(자릿수 2개)는 8진수의 1자리(자릿수 1개)와 같다. &nbsp; <- &nbsp; $2^3 = 8^1$
    - 2진수의 4자리는 16진수의 1자리와 같다. &nbsp; <- &nbsp; $2^4 = 16^1$
